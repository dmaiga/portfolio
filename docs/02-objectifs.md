# 02 — Objectifs

Trois objectifs, par ordre de valeur. Réussite mesurée par signal observable, pas par trafic.

1. **Rendre la méthode visible (prioritaire).** Depuis chaque réalisation, on atteint ses livrables de cadrage (artefacts `rules/`, `docs/`, `sprints/`) en **≤ 3 clics**, avec titres et contenus clairs, concis, sans bavardage — les fichiers qu'on attend d'un projet informatique sont là.
2. **Compréhension autonome.** Un pair ou recruteur saisit un projet **sans explication orale de ma part** : le site est intuitif, le projet se lit seul. Mesure non binaire : les retours de vive voix.
3. **Aide-mémoire.** Je retrouve le contexte d'un projet (quoi, pourquoi) en **< 30 s** via les docs qui lui sont liés.
